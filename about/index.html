About

This docs site contains varous documents relevant to Virga and includes tutorials, FAQs, and other references.

These docs are maintained by the Data Science Foundations Team.

Contributing

The source for these docs are hosted at github/virga/documentation-site. We welcome contributions.

Changes to this documentation are automatically deployed to Virga using self-hosted actions that are also running on Virga. You can see the Actions workflow that deploys these docs here.

Adding A New Doc

It can be helpful to look at the docs folder for examples. To create a new doc, write your documentation in GitHub flavored markdown, and place the appropriate front-matter the top of your markdown file. A minimal example of front matter is below:

---
layout: default
title: Service Level Agreement
nav_order: 3
---

More information about the front matter options as well as other customizations are illustrated in these docs, especially the section on navigation structure.

Editing A Document

To Edit a document make changes to the markdown in an existing markdown file in the docs folder.
